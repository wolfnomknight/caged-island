A Ilha Enjaulada - The Caged Island

A Ilha Enjaulada é um pequeno jogo baseado em texto, feito para praticar alguns conceitos de Python. Originalmente escrito no Replit.com: https://replit.com/@NomKnight/caged-island-project

The Caged Island is a small text-based game, made to practice some Python concepts. Originally wrote on Replit.com: https://replit.com/@NomKnight/caged-island-project
